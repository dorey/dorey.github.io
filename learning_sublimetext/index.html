Learning a new text editor

Over the next two weeks, I will learn a new text editor: sublimetext. I know that it's not hard to learn a new text editor, but I think if I force myself to dig deeper and try out some of the many plug-ins, then I will not stop once I get proficient, and I will be better off.

I will learn a new feature every day and post what I've learned. I'll pull from videos, tutorials, and anything else I find useful. If anyone has any features you recommend I try out, then please share your suggestions via twitter or email.

Why SublimeText?

I need to break my dependency on TextMate and OS X so that I can start using the linux machine that has been gathering dust for a couple months. My proficiency with TextMate has been the number one reason I've been crawling back to my old Mac. SublimeText is a respectable editor that has a lot of the benefits that TextMate has (including use of of 'tmbundles'.) It also runs in all the main operating systems.
